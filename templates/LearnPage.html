PhiMass Software

PhiMass Software is a versatile toolkit designed to assist researchers working with mass spectrometry and liquid chromatography data. It provides a comprehensive suite of solutions, ranging from reading raw data files generated by acquisition instruments to predicting the approximate retention time (RT) for any molecule.

Bruker (.d) File Converting

Leveraging ProteoWizard msconvert package to convert the Bruker .d Raw data folder to the standard data format for mass spectrometer output. In this step, CWT peak picking algorithm, and also Denoising, Deisotoping, and other filters applied.

Show Experiments

In This Page, You can show the Total ion chromatogram, Base Peak Chromatogram, and the Mass Spectrum for the Experiment that already converted to mzML file format. The Chromatograms are Interactive.

Compoundation of Features

In This Page, You Can Choose The Experiments (as mzML files) you plan to analyze, and apply xcms algorithms for Peak Picking, RT Alignment ..etc. The Features Table from this process is saved as csv file.

Annotation and Analysis

Empowering researchers to annotate features in the Features table and conduct advanced analyses, including Principal Component Analysis (PCA), Hierarchical Clustering, and Lipid Analysis.

Food Libraries

Search our database for any food item and download comprehensive CSV files containing all relevant data, including molecular information and concentration levels across multiple samples.

Compound Information

Search for compounds by name or chemical formula, and access detailed data from multiple references. You can also search for lipids by name or abbreviation.

Retention Time Prediction

Predict the retention time of any molecule under specific liquid chromatography conditions using our RT Index Model, powered by machine learning and scikit-learn for regression.

Publications

As every step in this software is scientifically backed and referenced, the data from metabolomics experiments could be published as a paper. For further information about citations, contact AbuZubaid.
